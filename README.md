# web-app_machine-learning_streamlit
Web Application pour la prévision des fleurs d'Iris

Dans ce projet on va s’intéresser à la classification des fleurs iris. Pour cela nous nous intéressons qu’aux trois catégories des fleurs de l’iris (Iris Versicolor, Iris Setosa, Iris Virginica), juste à partir des dimensions des feuilles. Donc là on a deux types de feuilles : les feuilles les plus petites appelées Petal et les feuilles les plus longues appelées Sepal. 

On veut que l’application puisse prévoir selon la dimension de Petal et Sepal est ce que la fleur est Versicolor ou Setosa ou bien Virginica.

Une fois le modèle est fini, j’ai utilisé streamlit qui va essayer de transformer ce modèle de machine learning en quelque chose de palpable. Elle va donc mettre en pratique ce modèle de machine learning.

Accédez à l’URL : http://localhost:8501
